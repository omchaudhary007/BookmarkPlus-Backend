# Bookmark+ Backend

Backend API for Bookmark+ with authentication, bookmark management, reminders, email verification, and in-app notifications.

---

## Features

- User signup, login, logout
- Email verification
- Password change
- Bookmark create, update, delete
- Categories, tags, priority, and status
- Reminder system (once, daily, weekly, monthly)
- Email reminders using Resend
- In-app notifications
- Global and per-bookmark reminder control
- Cron-based reminder processing

---

## Tech Stack

- Node.js
- Express
- TypeScript
- MongoDB (Mongoose)
- JWT Authentication
- Resend (Email Service)
